Installation
===================================

Installation on an existing site:
::

    composer require 'drupal/basket:^2.0.2@RC'
    

Additional libraries will be downloaded during installation:
::

    "require": {
        "mpdf/mpdf": "^8.0",
        "phpoffice/phpspreadsheet": "^1.6"
    }
